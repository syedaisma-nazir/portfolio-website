Water Potability Prediction Using Machine Learning

This project focuses on predicting whether water is safe for drinking by applying Machine Learning classification algorithms. The dataset was preprocessed, cleaned, and evaluated using multiple supervised learning models to compare their performance.

Project Overview

The aim of this project is to classify water samples as potable or non-potable based on various physicochemical properties. Different machine learning models were trained and evaluated to determine the most accurate prediction model.

Dataset Information

  • Total Samples: 3276
  • Input Features: 9
  • Missing Values: Filled using Median Imputation
  • Data Normalization: Z-Score Normalization
  • Train-Test Split: 80% Training / 20% Testing

Machine Learning Algorithms

  • Random Forest
  • Support Vector Machine (SVM)
  • Decision Tree
  • Logistic Regression
  • K-Nearest Neighbors (KNN)
  • Naïve Bayes

Project Results

Algorithm Accuracy
Random Forest 74%
Support Vector Machine 71%
Decision Tree 70%
KNN 68%
Logistic Regression 66%
Naïve Bayes 64%
