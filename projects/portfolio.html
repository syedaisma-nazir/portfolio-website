Portfolio Website

The Portfolio Website is a responsive personal website developed to showcase my education, technical skills, projects, resume, and contact information. The project demonstrates my front-end web development skills and serves as an online portfolio for internships and job opportunities.

Project Overview

This project was created to build a professional online presence where employers, recruiters, and collaborators can learn about my background, technical expertise, and completed projects. The website follows modern web development practices with a responsive design and interactive user interface.

Project Objectives

  • Create a professional personal portfolio website.
  • Showcase technical skills and academic projects.
  • Provide an online resume for employers.
  • Improve front-end web development skills.
  • Practice responsive web design techniques.
  • Apply version control using Git and GitHub.

Technologies Used

  • JavaScript (ES6)
  • Flexbox
  • CSS Grid
  • Responsive Web Design
  • Git
  • GitHub
  • GitHub Pages

Key Features

  • Responsive design for desktop, tablet, and mobile devices.
  • Professional Home and Hero section.
  • About Me section.
  • Skills section with technical expertise.
  • Projects section with detailed project pages.
  • Experience and Education sections.
  • Resume download functionality.
  • Professional contact form with validation.
  • Social media links.
  • Smooth scrolling, animations, and hover effects.
  • Scroll-to-top button.
