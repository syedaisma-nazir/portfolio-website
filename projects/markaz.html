Markaz E-Commerce Website

The Markaz E-Commerce Website is a responsive web application developed to provide users with an easy and modern online shopping experience. The project focuses on displaying products, managing product information, and creating a user-friendly interface using modern web technologies.

Project Overview

This project was designed to strengthen my front-end and back-end web development skills. It includes a responsive user interface, product management functionality, and database integration for storing and retrieving product information.

Objectives

  • Develop a responsive e-commerce website.
  • Display products in an organized layout.
  • Create reusable components for better code management.
  • Connect the website with a MongoDB database.
  • Implement RESTful APIs using Express.js.
  • Practice full-stack web development concepts.

Technologies Used

  • HTML5
  • CSS3
  • JavaScript (ES6)
  • React.js
  • Node.js
  • Express.js
  • MongoDB
  • Git & GitHub

Key Features

  • Responsive website design
  • Modern homepage layout
  • Product listing section
  • Dynamic product data
  • REST API integration
  • MongoDB database connectivity
  • Clean and user-friendly interface
  • Reusable React components

My Contribution

  • Designed the website interface using HTML and CSS.
  • Developed responsive layouts using Flexbox and CSS Grid.
  • Built frontend components using React.
  • Created backend APIs using Node.js and Express.js.
  • Integrated MongoDB for product data storage.
  • Tested and improved website responsiveness.
