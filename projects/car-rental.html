Car Rental System

The Car Rental System is an Object-Oriented Programming (OOP) based application developed to simplify the process of renting vehicles. The system enables customers to browse available cars, rent vehicles, return rented cars, and manage customer information while demonstrating the practical implementation of OOP principles.

Project Overview

This project was developed as part of an Object-Oriented Programming course to demonstrate software design using classes, objects, inheritance, encapsulation, polymorphism, and abstraction. The system automates the vehicle rental process, making it more organized, efficient, and user-friendly.
