Online Banking System

The Online Banking System is an Object-Oriented Programming (OOP) based application developed to simulate basic banking operations. The system enables users to create bank accounts, deposit and withdraw funds, transfer money, and check account balances while demonstrating the practical implementation of OOP principles.

Project Overview

This project was developed as part of an Object-Oriented Programming course to demonstrate the use of classes, objects, inheritance, encapsulation, polymorphism, and abstraction in solving real-world banking problems. The application provides a secure and organized way to perform common banking transactions.

Project Objectives
